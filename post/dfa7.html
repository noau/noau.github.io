【记录】个人博客的搭建

前言

国庆放假在家，没什么干的，就想着搭一个博客来玩玩。本来打算用 WordPress 的，但是 WP 不太好用，就换了 Hexo 。

成品: 戳这里

安装Hexo

主要就两条命令：
npm install hexo-cli -g
hexo init <folder>
就可以搭建好一个简单的 Hexo 博客。

配置基本信息

编辑博客目录下的 _config.yml 文件，没什么特殊的，主要是在 deploy 下配置 github 信息，到后面用 github pages 的时候就不用再配置了。

ps. Gitee 等也是在这里配置

配置主题

主题配置起来比较麻烦，花了我一天半的时间才折腾个大概。
配置了好多次主题，第一次用的是 mellow ，但是不是很喜欢，第二次的 melody 也是，icarus 不知道为什么运行不起来,最后用了现在这个配置相当简单的 volantis 。

volantis 配置起来很简单，跟着官方的教程来就很不错，全中文文档。主要是现在更新了 4.0.0 ，和网上查到的老版本会有很多不一样。

最麻烦的就是 blog/_config.volantis.yml 和 blog/themes/volantis/_config.yml，大部分教程（包括官方文档），都没怎么提到这两个的区别，折腾了一两个小时才搞明白。

（其实现在好多主题都在用 blog/_config.<themename>.yml 来配置主题，但是网上查到的文章有很多都太老，压根不会提到，有些主题又默认删掉了 blog/themes/<themename>/_config.yml 的文件，就会很迷。）

评论系统

这一部分我个人认为是最麻烦的，第一次搭建博客极其容易搞出一堆 Error 和 Warn. volantis 本身支持很多个评论系统，最开始打算用 gitalk ,但是各种报错，最后换成了 vssue 。
重点就在于多试几个，多重复几次，总有一次能成功的。
